# 📊 QDII 基金申购限额自动化监控助手 (fund_limit_monitor)

这是一个基于 Python 编写的 QDII 基金（主要针对纳斯达克100和标普500联接基金）单日申购限额监控工具。它可以自动从天天基金网抓取最新的交易状态与限额信息，对比个人设定的定投金额，并在工作日定时通过**企业微信机器人**推送格式化的 Markdown 播报。

## ✨ 核心功能

- 🔍 **数据自动抓取**：定时解析天天基金网公开数据，获取精确的基金交易状态及单日申购限额。
- 💰 **定投额度预警**：支持配置个人目标定投金额（`target_amount`），当基金限额低于你的定投需求时，自动打上 `⚠️` 标签提醒。
- 💬 **企业微信推送**：无缝对接企业微信群机器人，支持美观的 Markdown 格式排版（按“可申购/不可申购”及“指数类型”分类聚合）。
- ⚙️ **配置热更新**：无需重启服务，修改 `config.json` 即可实时生效。
- 📅 **智能定时任务**：内置自动化调度，仅在**周一至周五（工作日）**进行数据抓取与播报，周末自动跳过。
